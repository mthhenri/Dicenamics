Bem vindo ao Dicenamics!
Qual o propósito desse site?

Dicenamics é uma aplicação feita para
os jogadores e mestres de RPG que necessitam
de uma ferramenta de dados para seus RPGs, tanto os
online, quanto os presenciais, dados são a alma do RPG
e é isso que nos propomos a fazer: Rolar dados.


 Porquê devo usar o Dicenamics?

Além de ser uma aplicação completa, com
rolagem dados, customização dos dados, faces ilimitadas
e salas para rolagem com amigos, o Dicenamics é
COMPLETAMENTE BRASILEIRO.
ou seja, tudo aqui está em Português Brasileiro.


 Quais as funcionalidades?

Como lista de funcionalidades temos:

- Rolagem de Dados Customizados.
- Criação de Salas de Rolagem além de Dados Únicos de Salas.
- Diferentes Modos de Rolagem.
- Completamente Brasileiro.
